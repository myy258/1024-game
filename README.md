# 1024 game (compact interface)
## 1024 game introduction
- 4x4 grid, target 1024
- Direction keys / WASD control
- Can undo (max undos is 3)
- Advanced objective: The default objective of the game is 1024; When reaching 1024 for the first time, a pop-up prompt will appear and the target will be upgraded to 2048
